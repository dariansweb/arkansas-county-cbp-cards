DYS CBP US Census Dashboard

How to Use the Dashboard

  • Select a Year – View service totals by provider for any year between 2020–2024.
  • Filter Mode – Choose to filter by gender or age group, not both.
  • Gender Filter – Select "Male", "Female", or "Total" to filter provider data.

    Provider Selection: On desktop, hold Ctrl (or Cmd on Mac) to select multiple providers. On mobile, simply tap to select or deselect multiple entries.

  • Age Group Filter – Choose an age like 13, 16, or group like "12 and younger".
  • Provider Filter – Select one or more providers to show only their data.
  • Reset Button – Returns everything back to default (Gender mode, all providers, Year 2020).
  • County Cards – Show each county a provider serves, with:
    • Total Served: Full count from the provider (not split between counties)
    • Census Data: Estimated youth population for the selected age group
    • Breakdown: Male/Female population, and source group (10–14 or 15–19)

Note: Gender is ignored when age is selected, as provider data does not contain age-by-gender breakdowns.
